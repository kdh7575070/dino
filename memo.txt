---------------------------

1. startEngine
: 전반적인 게임 흐름을 담당함
: while문을 통해 반복하며 지속적으로 체크게임을 통하여 게임이 끝났는지 여부를 판단함
: score에 따라 밤 낮이 바뀌도록 설정되어 있음
: 스페이스바를 누를 시 공룡이 점프하는데, Y좌표를 7만큼 업 시켜줬다가 내려주는 방식임

2. computePrize
: 보상(불을 쏘는 기능) 개수를 계산하는 것 같음!
->이것보다는 코인? 혹은 조금 더 명확하게 이용하게 하면 좋을 것 같음!

3. endGame
: 현재 스코어가 최고 스코어를 넘으면 하이 스코어로 기록
: 공룡 캐릭터가 빨강으로 바뀌며 죽음
: 게임을 계속 할지 끝낼지 결정하게 함
-> 현재 10위까지 점수와 비교해서 파일을 바꾸어주는게 좋을 것 같음!

4. showDinasour
: 캐릭터가 움직이도록 해줌

5. startMenu
: 최고점수 파일을 불러옴, 이름을 받음, 나이를 받음 -> startEngine
->최고점수를 1-10등까지 저장해서 스코어 보드를 초기 화면에서 보여주는것도 괜찮을 것 같음!

6. computeTime
: 게임 속도를 점점 빠르게 만들어준다(?) -> 작동 하는건지는 체감이 잘 안되는 것 같음

7. checkGame
: 장애물의 위치와 캐릭터의 위치를 비교하여 게임을 종료할지 판단
